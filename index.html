Social KPIs Monitoring

A digital system to efficiently track and monitor children's development milestones.

We digitized the process using Google Forms and Big Query for data upload and storage, and developed an interactive dashboard in Power BI to visualize child development milestones.

Impact:

  • 80% reduction in data entry errors

  • Reports generated in seconds instead of days

  • Increased system adoption by the internal team

  • Faster, data-driven decision-making for early childhood support

View Project

Engagement Data Storing and visualization

A centralized system to automate and manage engagement data.

We centralized and cleaned the data, built a local data warehouse using DuckDB, and automated integration using the Google Drive API and Apache Airflow. An interactive dashboard was also developed to visualize participation and support decision-making.

Impact:

  • Consolidated data from over 90 forms

  • Significantly improved data quality and consistency

  • Fully automated data workflows with scalable architecture

  • Real-time insights through interactive dashboards

View Project

Analysis and Dashboard of production data

Sales, production and billing data were fragmented across departments, with no standardized input process, making analysis and decision making difficult.

We implemented a unified data entry process using Google Sheets, ensured consistent data formatting and connected the system to a customized dashboard in Looker Studio tailored to the KPIs.

Impact:

  • Centralized management of key business data.

  • Error reduction through standardized data entry.

  • Real-time information through a customized dashboard in Looker Studio

  • Improved cross-departmental efficiency and decision making.
